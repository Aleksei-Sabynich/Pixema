## 🚀 Быстрый запуск (Инструкция) 
https://pixema-2fc7.vercel.app

теперь API работет только через vpn, также блокируется запрос на авторизацию
### 1. Клонирование и переход 
 Откройте терминал и выполните:
    bash
    git clone https://github.com/Aleksei-Sabynich/Pixema.git
    cd Pixema
### 2. Установка зависимостей
  Скачайте все необходимые библиотеки (создаст папку node_modules):
    bash
    npm install
### 3. Запуск проекта
  Запустите сервер для разработки:
    bash
    npm run dev
  Проект откроется по адресу: http://localhost:5173
### 4. Сборка для продакшена
  Для создания готовых оптимизированных файлов в папке dist:
    bash
    npm run build

### 📂 Команды в терминале (Bash)
    CTRL + C — остановить работающий сервер в терминале.
